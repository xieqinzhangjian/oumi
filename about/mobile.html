购买常识

什么是白菜价？

白菜价是指白菜网上所销售的商品的即时价格；我们实行线上、线下统一售价，统称为白菜价。



什么是正品行货？

主要是指中国生产，销售于中国市场的产品一般称之为正品行货。可享受到在所设的客服中心免费保修及三包服务。由于受厂家销售策略及政府要求的关税和税款的影响，一般价格都会略高于其它货源。由于受国家政策影响，通常会削减一些功能，如：4G网络等。但相较于其他货源，购买正品行货是保障自身利益的最明智选择。



是否原封未拆封？

正品行货机型到货均是厂家原封未拆封的，您可以放心选购。


使用常识

日常使用中如何保养手机？

所谓"预防胜于治疗"，如果每次都等到手机出事再补救，可能已经晚了！以下有5个容易用坏手机的例子，全部都是经常犯的小毛病，如果可以避免，就可以使你的手机处于安全状态。
忌一：避免触及SIM卡的金属片，任何刮毁或损伤，都会引致SIM卡无法读取。
忌二：切忌将手机放在冷气的出风口，因为凝结在手机的水气会无形地腐蚀机板。
忌三：充电时间要适可而止，一般8小时己足够，若过量充电反而会减短电池寿命。
忌四：谨记，手机不要与硬物如钥匙等物品装在一起，磕碰刮损会加速手机的磨损。
忌五：为怕手机意外沾湿，套上防水袋及防水胶囊，可以起到保护之效。


手机进水如何紧急抢救？

相信大家都遇到过手机意外进水的情况，出现这种情况正确的处理方法是不要迟疑，立即关掉电源，并将电池取出，以避免水滴腐蚀机板。

3G热点问题解答

什么是3G？3G有哪些优势？
3G全称为3rd Generation，中文含义就是指第三代数字通信。3G目前有3种标准，分别是欧洲的WCDMA制式，美国的CDMA2000制式和中国自主研发的TD-SCDMA制式，3G与现在使用的2.5G最大的优势是视频通话和媲美宽带的网速。


中国3大运营商都采用哪种3G网络制式?

TD-SCDMA：中国移动运营，自主研发，目前还只有在中国正式商业运营。
WCDMA：中国联通运营，目前最成熟使用人数最多的3G标准，目前在欧洲最高速度可以达到14.4Mbps。
CDMA2000：中国电信运营，成熟度仅次于WCDMA。目前被美国、韩国、日本等国家采用。


享受3G是否必须更换3G手机和3G手机卡？

TD-SCDMA（中国移动）：必须购买一部支持TD-SCDMA的手机，购买新的USIM卡更换TD专用号段（157、187、188号段）。现在公布的消息是必须办理TD-SCDMA的专用号段，以后可以实现运营商之间的携号转网。
WCDMA（中国联通）：由于WCDMA标 准已经被很多国家采用，所以已经有了很多手机都支持WCDMA。简单来说，大部分手机都支持WCDMA，所以想要加入WCDMA阵营的用户只 需要更换一张USIM卡即可。当然，前提条件是现在使用的手机支持WCDMA。中国联通支持不换号使用3G服务，当月办理下月生效，必须到营业厅办理。
CDMA2000（中国电信）：必须购买一部支持CDMA2000的手机以及购买一张USIM卡。
智能手机操作系统


Android 操作系统：
Android一词的本义指“机器人”，同时也是Google于2007年11月5日宣布的基于Linux平台的开源手机操作系统的名称，该平台由操作系统、中间件、用户界面和应用软件组成，号称是首个为移动终端打造的真正开放和完整的移动软件。Android 作为谷歌的重要操作平台是要确保每一个用户在自己喜爱的电话上安装谷歌手机地图、Gmail 以及其他一些专为你的手机开发的精彩应用。
优点：
开放性，挣脱运营商的束缚。
丰富的硬件选择。
不受任何限制的开发商。
无缝结合的Google应用。
缺点：
Android系统手机泄密 信息时代很严重。
拨号后自动挂断电话 通话BUG频繁出现。
对硬件配置要求高 制造成本增加。
系统费电严重 安卓手机续航不足，系统计算器计算有偏差。

苹果iOS 操作系统：
iPhone OS 或 OS XiPhone是由苹果公司为iPhone开发的操作系统。它主要是给iPhone和iPodtouch使用。iPhoneOS的系统架构分为四个层次：核心操作系统层（the Core OSlayer），核心服务层（the Core Serviceslayer），媒体层（the Media layer），可轻触层（theCocoa Touchlayer）。系统操作占用大概512MB的存储空间。
iPhone OS由两部分组成：操作系统和能在iPhone和iPod touch设备上运行原生程序的技术。
优点：
人性化的操作。
系统的深度优化和数以百万的应用程序支持。
iOS平台的优化性。
最优秀的智能系统。
缺点：
价格过高。
续航不强、不能换电池等问题。

Windows Phone 8 操作系统：
Windows Phone 8是微软公司2012年6月21日最新发布的一款手机操作系统，是Windows Phone系统的最新版本，也是Windows Phone的第三个大型版本。Windows Phone 8采用和Windows 8相同的针对移动平台精简优化NT内核并内置诺基亚地图。诺基亚与微软的合作正在逐步加深。
优点：
系统运行非常流畅，很少出现死机的情况。
动态磁贴的桌面设计非常新颖，个性十足。
内置Office办公软件，非常适合移动办公。
文件兼容性佳。
缺点：
系统封闭，可玩性差。
支持机型有限，外观雷同，可选性较差。
目前，应用扩展性暂时无法与iOS和Android抗衡。
